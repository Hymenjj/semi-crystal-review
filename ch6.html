第六章

半导体中的点缺陷

零维缺陷：空位、间隙原子、杂质。它们怎么产生、浓度多少、如何让材料变 N 型或 P 型——这是半导体掺杂的微观基础。

🎯 本章要点

点缺陷分类与克罗格-文克（克文）符号；三种热缺陷模型（弗兰克、肖特基、第三种）及反应方程；缺陷平衡浓度（\(n=N e^{-u/kT}\) 类）；杂质（替位式 / 间隙式）与韦伽定律；化合物 MX 的点缺陷（含反结构缺陷）；点缺陷电离与施主/受主作用；化学计量比偏离与导电类型；外压控制；点缺陷缔合。

🧭 一句话脉络

晶体不是完美的，总有缺位（空位）、挤进去的原子（间隙原子）、外来原子（杂质）。用克文符号给每个"哪儿是什么"命名 → 写出缺陷的生成反应 → 算出浓度（温度越高越多）→ 缺陷电离后给出电子或空穴，决定材料是 N 型还是 P 型。

6.1 点缺陷与克文符号

点缺陷又称零维缺陷，偏离点阵的部位只是一个或几个粒子。四种基本点缺陷 + 它们的复合体。克罗格-文克符号（克文符号）用"微粒名称 + 位置下标"描述每个位置：

符号含义
\(M_M\)正常格点上的基体原子 M
\(M_s\)表面格点上的 M（surface）
\(V_M\)M 位置的空位（Vacancy）
\(M_i\)间隙原子（interstitial）
\(F_M\)替位式杂质（外来 F 占 M 位）
\(F_i\)间隙杂质（外来 F 在间隙）

6.1.1 三种热缺陷模型（核心）

本征点缺陷 = 热缺陷：由晶格热振动产生，浓度依赖温度。它与外来杂质引入的"原生缺陷"要区分开。三种基本模型：

⭐ 弗兰克 / 肖特基 / 第三种（必背反应方程）
模型机制反应方程
弗兰克缺陷 (Frenkel)空位与间隙原子成对产生\(M_M \rightleftharpoons M_i + V_M\)
肖特基缺陷 (Schottky)只产生空位（原子迁到表面）\(M_M \rightleftharpoons M_s + V_M\)
第三种缺陷只产生间隙原子\(M_s \rightleftharpoons M_i\)

6.1.2 热缺陷浓度

把晶体当溶液体系（正常格点=溶剂，点缺陷=溶质），热缺陷看作可逆准化学平衡，用质量作用定律写平衡常数 \(K\)。或用统计热力学：平衡时自由能 \(F\) 最小。以肖特基（只有空位）为例：

\[ \Delta F=n\,u_1-T\Delta S,\qquad S=k\ln W,\qquad n=N\,e^{-u_1/kT} \]

式中 \(N\) 为原子数、\(u_1\) 为一个空位的形成能。弗兰克缺陷类似可得 \(n=\sqrt{NN^*}\,e^{-E_F/2kT}\)（\(E_F\) 为一个空位-间隙对的形成能）。

🔑 两条核心结论
  1. 形成能越大，缺陷平衡浓度越小（指数里 \(-u/kT\)）。
  2. 温度越高，缺陷浓度越大。

过饱和空位：高温空位多，若快速降温（淬火），大量空位来不及扩散复合就被"冰结"在体内，形成超过平衡值的过饱和空位（此时不满足热力学平衡）。

6.1.3 杂质原子

外来杂质按位置分两类：

  • 替位式杂质：占据正常格点。Si、Ge 中掺 III 族（B、Al、Ga、In）或 V 族（P、As、Sb）——结构与 IV 族相似，易形成替位式，用于控制导电类型和电阻率。
  • 间隙式杂质：挤入间隙。Fe、Ni、O 与 IV 族相差较多，易形成间隙式。Au、Cu 则两者都可能。
替位式杂质 vs 替代式固溶体 & 韦伽定律

具有相同价电子数的替位杂质晶体才称固溶体（如 \(\text{Si}_{1-x}\text{Ge}_x\)）。韦伽定律：\(a=(1-x)a_1+x\,a_2\)，替位式杂质引起的晶格畸变也可近似用它估算。半径大的替位杂质使晶格膨胀，小的使晶格收缩。

6.2 化合物半导体中的点缺陷

化合物 MX（M=电负性小的金属，X=电负性大的非金属）点缺陷更复杂。全部可能的缺陷：

类别缺陷
弗兰克缺陷成对的 \(V_M\) 和 \(M_i\)，成对的 \(V_X\) 和 \(X_i\)
肖特基缺陷\(V_M\) 或 \(V_X\)（生成能低，几率和浓度大）
第三种缺陷\(M_i\) 或 \(X_i\)（生成能高，一般不考虑）
反结构缺陷成对的 \(M_X\) 和 \(X_M\)（原子占了对方的位置，antisite）
杂质缺陷替位 \(F_M\)、\(F_X\)，间隙 \(F_i\)

注意：晶体生长过程是电中性的，缺陷是原子而非离子。

6.2.2 点缺陷的电离：施主与受主（高频）

⭐ 哪些缺陷是施主、哪些是受主（背这张表）
产生施主能级（给出电子，→N 型）产生受主能级（接受电子，→P 型）
\(M_i\)（金属间隙原子）\(X_i\)（非金属间隙原子）
\(V_X\)（非金属空位）\(V_M\)（金属空位）
\(X_M\)（非金属占金属位，反结构）\(M_X\)（金属占非金属位，反结构）
\(F_i\)（F 为金属元素）\(F_i\)（F 为非金属元素）
\(F_M\)（F 价 \(\gt\) M 价）\(F_M\)（F 价 \(\lt\) M 价）

口诀：非金属空位 \(V_X\)、金属间隙 \(M_i\)、非金属占金属位 \(X_M\) → 施主；反过来 \(V_M\)、\(X_i\)、\(M_X\) → 受主。

对 III-V 族弱极性共价键晶体：\(V_M\) 空位缺 3 个价电子、\(V_X\) 空位缺 5 个价电子，都倾向从价带获取电子起受主作用（GaAs 等行为更复杂）。

6.2.4 化学计量比偏离（重点）

化合物理论上按整数比（化学计量比）结合，但实际多少有偏离。

⭐ 偏离方向 → 导电类型
  • 正偏离（X 组分过量、M 不足）：\(X_i\)、\(X_M\)、\(V_M\) 过量 → 起受主作用 → P 型。
  • 负偏离（M 组分过量、X 不足）：\(M_i\)、\(M_X\)、\(V_X\) 过量 → 起施主作用 → N 型。

一句话：M（金属）过量 → N 型；X（非金属）过量 → P 型。

哪种晶体易偏离？离子键成分强的晶体——结合不受饱和性、方向性限制——更容易出现化学比偏离；共价键成分强的（有方向性、饱和性）偏离度小。许多 II-VI、III-V 化合物（PbS、ZnS、CdS）合成后不掺杂就强烈呈 N 型或 P 型，难做 P-N 结。

外压（蒸气压）控制——PbS 例子

固定温度 \(T\)，改变非金属组分蒸气压 \(P_{X_2}\) 可调节缺陷浓度与导电类型。PbS：当 \(P_{S_2}\lt100\,\text{Pa}\) 为 N 型（Pb 过量）；\(P_{S_2}\gt100\,\text{Pa}\) 为 P 型；\(P_{S_2}\approx100\,\text{Pa}\) 时为化学计量比 1:1 的近本征 PbS。

6.3 点缺陷的缔合

带电点缺陷并非杂乱分布，相邻缺陷可借库仑引力、偶极矩、共价键、弹性作用等相互缔合成缔合体。温度升高（热运动加剧）缔合体会重新分解。例：纯 ZnS 不发光，加入 Cu、Ag、Au 才发光，再加 Br⁻、Cl⁻（共激活剂）发光增强。点缺陷可形成深能级、色心、载流子陷阱、复合中心。

先盖住自己想。👇

例 1三种热缺陷的反应方程概念
题：用克文符号写出单质晶体 M 中弗兰克缺陷、肖特基缺陷、第三种缺陷的反应方程，并说明各产生什么缺陷。
1
弗兰克缺陷

\(M_M \rightleftharpoons M_i + V_M\)：一个格点原子跳到间隙，同时产生空位和间隙原子（成对）。

2
肖特基缺陷

\(M_M \rightleftharpoons M_s + V_M\)：原子迁移到表面，只产生空位，不产生间隙原子。

3
第三种缺陷

\(M_s \rightleftharpoons M_i\)：表面原子进入间隙，只产生间隙原子。

答：弗兰克=空位+间隙成对；肖特基=只空位；第三种=只间隙。三者都是热缺陷，浓度随温度升高而增大。
例 2化合物 MX 中的全部点缺陷类型概念
题：列出化合物晶体 MX 中可能出现的全部点缺陷（用克文符号），并指出哪种热缺陷生成能最低、占主导。
1
热缺陷

弗兰克：成对的 \(V_M,M_i\) 和成对的 \(V_X,X_i\)；肖特基：\(V_M\) 或 \(V_X\)；第三种：\(M_i\) 或 \(X_i\)（生成能高，一般不考虑）。

2
反结构缺陷

成对的 \(M_X\) 和 \(X_M\)（原子占了对方位置，antisite）。

3
杂质缺陷

替位 \(F_M\)、\(F_X\)，间隙 \(F_i\)。

4
主导

一般认为只产生空位的肖特基缺陷生成能较低，形成几率和浓度最大。

答：热缺陷（弗兰克/肖特基/第三种）+ 反结构缺陷 \(M_X,X_M\) + 杂质缺陷；肖特基占主导。
例 3判断点缺陷的施主/受主作用概念
题：对二元化合物 MX，判断 \(V_X\)、\(V_M\)、\(M_i\)、\(X_i\)、\(X_M\)、\(M_X\) 各起施主还是受主作用。
1
非金属空位 \(V_X\)

拿走电中性非金属，留下束缚电子易游离成导带电子 → 施主。

2
金属空位 \(V_M\)

留下束缚正电荷（空穴）→ 受主。

3
间隙原子

金属间隙 \(M_i\)（电负性小、易放电子）→ 施主；非金属间隙 \(X_i\)（易捕获电子）→ 受主。

4
反结构缺陷

\(X_M\)（非金属占金属位，外层电子多、易放电子）→ 施主；\(M_X\)（金属占非金属位）→ 受主。

答：施主：\(V_X,M_i,X_M\)；受主：\(V_M,X_i,M_X\)。口诀：非金属空位/金属间隙/非金属占金属位 = 施主，反之受主。
例 4化学计量比偏离与导电类型（PbS）分析
题：PbS 一般制备出来为 N 型，为什么？如何通过控制硫蒸气压 \(P_{S_2}\) 改变其导电类型？
1
为什么天然 N 型

PbS 是离子键成分较强的化合物，易出现化学计量比偏离。制备时常 Pb（金属）过量（负偏离），产生 \(M_i\)、\(V_X\) 等施主缺陷 → N 型。

2
规律

M（金属）过量 → 施主 → N 型；X（非金属）过量 → 受主 → P 型。

3
用蒸气压调控

固定 \(T\)，改变 \(P_{S_2}\)：\(P_{S_2}\lt100\,\text{Pa}\) → S 不足、Pb 过量 → N 型；\(P_{S_2}\gt100\,\text{Pa}\) → S 过量 → P 型；\(P_{S_2}\approx100\,\text{Pa}\) → 1:1 近本征。

答：PbS 因 Pb 过量天然 N 型；提高硫分压使 S 过量可转为 P 型，\(\approx100\,\text{Pa}\) 时为化学计量比 1:1 的近本征材料。

点选项即时判分给解析；多选题选完点「提交答案」。

点缺陷又称
  • 零维缺陷
  • 一维缺陷
  • 二维缺陷
  • 三维缺陷
点缺陷=零维（偏离点阵的部位是一个/几个粒子）；位错=一维（线）；面缺陷=二维。【秒记】点=0维、线(位错)=1维、面=2维。
克文符号 \(V_M\) 表示
  • 间隙原子
  • M 位置上的空位
  • 替位式杂质
  • 表面原子
\(V_M\)=空位（Vacancy），\(M_i\)=间隙原子，\(F_M\)=替位杂质，\(F_i\)=间隙杂质，\(M_s\)=表面原子。【秒记】V=空位、i=间隙、F=外来杂质。
弗兰克（Frenkel）缺陷的反应方程是
  • \(M_M \rightleftharpoons M_i + V_M\)（空位与间隙成对）
  • \(M_M \rightleftharpoons M_s + V_M\)（只产生空位）
  • \(M_s \rightleftharpoons M_i\)（只产生间隙）
  • \(M_M \rightleftharpoons 2V_M\)
弗兰克=空位+间隙成对；肖特基=只产生空位（原子到表面）；第三种=只产生间隙。【秒记】弗兰克=空位+间隙成对。
肖特基（Schottky）缺陷的特点是
  • 只产生间隙原子
  • 只产生空位（原子迁移到表面）
  • 空位与间隙成对产生
  • 不依赖温度
肖特基缺陷 \(M_M\rightleftharpoons M_s+V_M\)，原子迁到表面、只留空位。化合物中肖特基生成能低，占主导。【秒记】肖特基=只空位（去表面）。
关于热缺陷平衡浓度 \(n=N e^{-u/kT}\)，正确的是
  • 形成能越大浓度越大
  • 形成能越大浓度越小，温度越高浓度越大
  • 与温度无关
  • 温度越高浓度越小
指数 \(-u/kT\)：形成能 \(u\) 越大 → 浓度越小；温度 \(T\) 越高 → 浓度越大。这是两条核心结论。【秒记】能量大浓度小，温度高浓度大。
高温空位多，快速降温使大量空位"冰结"在体内，形成的是
  • 反结构缺陷
  • 过饱和空位
  • 肖特基缺陷
  • 替位杂质
淬火（快速降温）时空位来不及扩散复合，被冰结成超过平衡值的过饱和空位，此时不满足热力学平衡。【秒记】淬火冰结=过饱和空位。
Si、Ge 中掺入 III 族或 V 族元素，一般形成
  • 替位式杂质
  • 间隙式杂质
  • 反结构缺陷
  • 空位
III、V 族结构与 IV 族相似，易占据格点形成替位式杂质，用于控制导电类型；Fe、Ni、O 等相差大，易形成间隙式。【秒记】III/V掺Si/Ge=替位式。
化合物 MX 中"原子占据对方格点位置"形成的缺陷称为
  • 肖特基缺陷
  • 弗兰克缺陷
  • 反结构缺陷（\(M_X\)、\(X_M\)）
  • 间隙杂质
反结构缺陷（antisite）：\(M_X\)（金属占非金属位）、\(X_M\)（非金属占金属位），是化合物特有的缺陷类型。【秒记】反结构=占了对方的位置。
下列点缺陷中起施主作用的是
  • 金属空位 \(V_M\)
  • 非金属空位 \(V_X\)
  • 非金属间隙 \(X_i\)
  • 金属占非金属位 \(M_X\)
施主：\(V_X\)（非金属空位）、\(M_i\)（金属间隙）、\(X_M\)（非金属占金属位）；受主：\(V_M\)、\(X_i\)、\(M_X\)。【秒记】施主=\(V_X,M_i,X_M\)。
化合物半导体中，金属组分 M 过量（负偏离化学计量比）时，材料一般呈
  • N 型
  • P 型
  • 本征型
  • 绝缘体
M 过量 → 产生 \(M_i\)、\(V_X\) 等施主缺陷 → N 型；X 过量 → 受主 → P 型。【秒记】金属过量N型，非金属过量P型。
下列哪类晶体更容易出现化学计量比偏离？
  • 离子键成分强的晶体
  • 共价键成分强的晶体
  • 金属晶体
  • 分子晶体
离子键无饱和性、方向性限制，组元比例不易固定，易偏离；共价键有方向性饱和性，偏离度小。PbS、ZnS、CdS 等易偏离、常天然 N 型。【秒记】离子键易偏离化学计量比。
三种热缺陷模型包括（多选）
  • 弗兰克缺陷（空位+间隙成对）
  • 肖特基缺陷（只空位）
  • 第三种缺陷（只间隙）
  • 反结构缺陷
单质晶体热缺陷三模型：弗兰克、肖特基、第三种。反结构缺陷是化合物里才有的（不属于这三种热缺陷模型）。【秒记】热缺陷三模型=弗兰克/肖特基/第三种。
下列点缺陷起受主作用的有（多选）
  • 金属空位 \(V_M\)
  • 非金属间隙原子 \(X_i\)
  • 金属占非金属位 \(M_X\)
  • 非金属空位 \(V_X\)
受主：\(V_M\)、\(X_i\)、\(M_X\)；\(V_X\) 是施主。【秒记】受主=\(V_M,X_i,M_X\)；施主=\(V_X,M_i,X_M\)。
关于点缺陷浓度与缔合，正确的有（多选）
  • 缺陷形成能越大，平衡浓度越小
  • 温度越高，热缺陷浓度越大
  • 带电点缺陷可借库仑力等相互缔合
  • 温度升高，缔合体倾向重新分解
浓度规律来自 \(n=Ne^{-u/kT}\)；缔合靠库仑、偶极、共价、弹性作用，升温热运动加剧使缔合体分解。【秒记】能大浓小、温高浓大；升温拆缔合。
