第一章

晶体的特性

什么是晶体、它凭什么和玻璃不一样、原子靠什么"粘"在一起。本章名词多但都好记，是后面所有内容的入门。

🎯 本章要点

晶体的定义与远程有序；晶体四大基本性质（自范性、均匀性与各向异性、对称性、最小内能与固定熔点）；电负性概念 \(X=0.18(I+Y)\)；五种典型键型（离子键、共价键、金属键、分子键、氢键）及其判据与特点；混合键型典型例子——石墨。

🧭 一句话脉络

晶体 = 内部微粒按规则周期排列的固体。这种"周期性"带来一系列特殊性质；而把微粒"粘"成晶体的力，按强弱和性质分成五种键型，键型又决定了材料是绝缘体、半导体还是金属。

1.1 晶体和非晶体

晶体的定义：内部微粒（原子、分子或离子）按一定规则周期排列而构成的固体，也叫"具有格子构造的固体"。1912 年劳埃（Laue）的 X 射线衍射实验第一次证明了晶体内部确实有这种周期性的格子构造。

💡 远程有序 vs 近程有序（最爱考的对比）

晶体——远程有序：内部粒子高度规律，在某方向的直线上规则重复千百万次。
非晶体——近程有序、远程无序：近似于液态，只在很小范围内有序，又称无定型体（如石英玻璃）。

概念含义
单晶整个晶体内部微粒都按同一规则周期性排列
多晶许多小块单晶组成，各小块大小、取向各不相同
纳米晶晶体尺度在纳米量级；原子数有限、表面原子比例很大 → 特异性质
液晶固→液之间的中间态，性质介于晶体和液体之间

1.2 晶体的基本性质（四条，必背）

1自范性▸

晶体能自发生长成结晶多面体，常以平面作为与周围介质的分界面。例：天然矿物的规则外形；拉单晶时放肩部位出现的平整小平面、等径部位的棱线，都是自范性的表现。

2均匀性 与 各向异性▸

均匀性：晶体各个部位的宏观性质完全相同。
各向异性：沿不同方向看，性质有差异。

例：云母片导热的各向异性；石墨不同方向电导率不同；Si、Ge、GaAs 等半导体的物理、生长、腐蚀特性沿不同方向都不同——这正是各向异性的表现（后面第五章重点）。

3对称性▸

所有晶体在外形和性质上都或多或少具有对称性（第三章专门讲）。

4最小内能 与 固定熔点▸

气/液/非晶 → 晶体要放热，反之要吸热，说明一定 \((T,P)\) 下晶态内能最小。

晶体有固定熔点：晶体内各结构单元大小构造完全相同，一个单元被破坏时其他单元在同一温度下也被破坏，于是在某一温度整体完成固→液转变，这个温度就是熔点。
非晶体无固定熔点：结构单元大小不一，破坏所需能量不同，升温时逐渐软化。非晶体可看作"过冷的液体"。

1.3 晶体的类型和结合力

电负性

为比较不同原子拉电子能力的强弱，引入电负性。它由原子的电离势能 \(I\)（放弃一个电子耗的能量）和电子亲合能 \(Y\)（得到一个电子放出的能量）组合表示：

电负性\[ X = K(I+Y), \qquad \text{取 } K=0.18\ \text{eV}^{-1}:\quad X=0.18(I+Y) \]

（无量纲；规定 Li 的电负性为 1 来定 \(K\)。）电负性差 \(\Delta X\) 是判断键型的关键。

🔑 用 ΔX 判断键型（高频考点）
  • 离子键：\(\Delta X \gt 1.5\)，价电子几乎全被电负性大的原子占有，形成正负离子。例：NaCl（Cl 的 \(X=3.0\)，Na 的 \(X=0.9\)）。
  • 非极性共价键：\(\Delta X = 0\)（同种原子）。例：金刚石、Si、Ge。
  • 极性共价键：\(0 \lt \Delta X \lt 1.5\)，共用电子对偏向电负性大的原子，含一定离子键成分。例：GaAs \(\Delta X=0.4\)、InSb \(\Delta X=0.2\)、GaP \(\Delta X=0.5\)。

五种典型键型

键型判据 / 机制特点例子
离子键\(\Delta X \gt 1.5\)，正负离子静电吸引无方向性、无饱和性；硬度大、熔点高、熔融后能导电；常溶于极性溶剂NaCl
共价键\(\Delta X \lt 1.5\)，共用电子对有方向性、有饱和性；硬度和熔点一般较高；常温纯净时不导电金刚石、Si、Ge、GaAs
金属键电负性小，价电子公有化成"电子气"良好导电性和延展性；趋向最紧密堆积金属单质（Au、Cu…）
分子键分子间范德华力作用力弱；硬度小、熔点低CO₂、I₂、干冰等
氢键含 –OH、–NH 或 HF，H 与 O/N/F 间静电吸引有方向性、有饱和性，作用力较弱；配位数低、密度小、熔点低冰、草酸、硼酸
🪙 打比方：方向性 / 饱和性

离子键像磁铁吸铁屑——四面八方都能吸，吸多少看空间（无方向、无饱和）。共价键像插头插插座——只能朝特定方向、插满就不能再插（有方向、有饱和）。所以共价晶体（Si）才有确定的四面体几何。

分子键的三种范德华力

  • 静电力（葛生力）：极性分子之间的相互作用。
  • 诱导力（德拜力）：极性分子与非极性分子之间。
  • 色散力（伦敦力）：非极性分子之间（电子云瞬间分布不均 → 瞬时偶极矩）。

杂化轨道与 Si 的四面体结构

成键时，同一原子能量相近的不同轨道可线性组合成数目相等的新轨道，叫杂化轨道，成键能力更强。Si 外层 \(3s^2 3p^2\) 的 4 个电子形成 4 个等同的 sp³ 杂化轨道，电子云指向正四面体四个顶角，键角 \(109°28'\)，沿四面体键共价结合 → 金刚石型结构（第五章细讲）。

1.3.6 混合价键型晶体：石墨

⭐ 石墨——三种键并存（经典考点）

同一种晶体里同时有多种键型，就叫混合键型晶体。石墨是典型：

  • 层内：C 原子 sp² 杂化，每个 C 与相邻 3 个 C 以共价键（σ 键）连成六角蜂巢平面 → 这部分是共价（原子）晶体，键长 142 pm。
  • 层内还有 p 电子：剩下一个 p 轨道相互重叠，电子较自由（类似金属自由电子）→ 石墨能导电导热，这是金属晶体特征。
  • 层与层之间：相距 340 pm，靠范德华力结合 → 这部分是分子晶体，所以石墨易滑移（做润滑剂）。

正因为这种结构，石墨有金属光泽、层平面方向导电好、易滑移、耐高温，是优良的固体润滑剂和高温电极。从石墨剥离出的单原子层就是石墨烯（二维晶体）。

例 1用电负性判断键型判断
题：已知 Cl 的电负性 \(X=3.0\)、Na 的 \(X=0.9\)、Ga 的 \(X=1.6\)、As 的 \(X=2.0\)。判断 NaCl 与 GaAs 各属于什么键型。
1
NaCl

\(\Delta X=3.0-0.9=2.1\gt1.5\)，价电子几乎完全转移给 Cl → 离子键，NaCl 是典型离子晶体。

2
GaAs

\(\Delta X=2.0-1.6=0.4\)，满足 \(0\lt\Delta X\lt1.5\) → 极性共价键（共用电子对偏向 As，含一定离子键成分）。

答：NaCl 离子键；GaAs 极性共价键。判据口诀：大于 1.5 离子，等于 0 纯共价，中间极性共价。
例 2晶体为什么有固定熔点，非晶体没有？简答
题：从内部结构解释：为什么晶体有固定熔点，而非晶体（如玻璃）只是逐渐软化？
1
晶体

晶体内部各结构单元大小、构造完全相同，破坏一个单元所需能量与其他单元相同。升温到某温度时，所有单元在同一温度同时被破坏，整体完成固→液转变 → 有固定熔点。

2
非晶体

非晶体没有规则格子，各结构单元大小、结构不尽相同，破坏所需能量不同。升温时一些单元先被破坏、另一些要更高温度，于是没有固定熔点，只是逐渐软化。可看作过冷液体。

答：根源在于晶体结构单元的一致性（周期性），使破坏集中在同一温度发生。
例 3石墨为什么是混合键型晶体简答
题：说明石墨晶体中存在哪几种键，并解释它"既能导电、又易滑移、又耐高温"。
1
层内 σ 键（共价）

C 原子 sp² 杂化，每个与 3 个相邻 C 形成共价键，连成六角蜂巢平面，结合牢固 → 耐高温、强度高。

2
层内 p 电子（金属性）

每个 C 还剩一个 p 轨道，p 电子相互重叠、比较自由，像金属自由电子 → 沿层平面导电导热。

3
层间范德华力（分子键）

层与层相距 340 pm，靠弱的范德华力结合 → 层间易滑移，做固体润滑剂。

答：石墨同时含共价键 + 金属键 + 分子键，故为混合键型晶体；三种键各自决定了它的不同性质。

晶体最本质的特征是
  • 有固定的几何外形
  • 内部微粒按一定规则周期性排列（远程有序）
  • 一定是透明的
  • 硬度很大
晶体的定义就是内部微粒周期性排列、具有格子构造，即远程有序。外形、硬度等都是周期性带来的表现，不是本质。【秒记】晶体=周期排列=远程有序。
非晶体（如石英玻璃）的结构特征是
  • 远程有序
  • 近程有序、远程无序
  • 完全无序，无任何规律
  • 比晶体更有序
非晶体近似于液态，只在很小范围内有序，又称无定型体，可看作过冷的液体。【秒记】非晶体=近程有序、远程无序。
第一次用实验证明晶体内部具有周期性格子构造的是
  • 伦琴发现 X 射线
  • 劳埃的 X 射线衍射实验（1912）
  • 戴维逊-革末实验
  • 康普顿散射
1912 年劳埃用 X 射线照射晶体得到规则衍射斑点，证明了晶体的周期性格子构造（同时也证明 X 射线是电磁波）。【秒记】劳埃 X 射线衍射=证明格子构造。
晶体"能自发生长成规则多面体外形"的性质称为
  • 自范性
  • 各向异性
  • 对称性
  • 均匀性
自范性指晶体自发生长成结晶多面体、常以平面为分界面，如天然矿物外形、拉单晶时的小平面和棱线。【秒记】自范=自己长成多面体。
"云母沿不同方向导热不同、石墨不同方向电导率不同"，体现晶体的
  • 均匀性
  • 各向异性
  • 自范性
  • 最小内能
各向异性指沿不同方向性质有差异；均匀性指各部位性质相同。两者不矛盾：同一方向处处相同（均匀），不同方向可不同（各向异性）。【秒记】各向异性=换个方向就不一样。
关于晶体的熔点，正确的是
  • 晶体有固定熔点，非晶体逐渐软化无固定熔点
  • 晶体和非晶体都有固定熔点
  • 非晶体有固定熔点，晶体没有
  • 都没有固定熔点
晶体结构单元相同，在同一温度同时被破坏 → 固定熔点；非晶体单元大小不一，逐渐软化。【秒记】晶体有固定熔点（结构单元一致）。
电负性的表达式（课程取法）为
  • \(X=0.18(I+Y)\)，\(I\) 为电离势能、\(Y\) 为电子亲合能
  • \(X=0.18(I-Y)\)
  • \(X=I\cdot Y\)
  • \(X=h\nu\)
电负性反映原子拉电子的能力，用电离势能与电子亲合能之和表示，取 \(K=0.18\) 使 Li 的电负性为 1。【秒记】电负性=0.18×(电离势+亲合能)。
两元素电负性差 \(\Delta X \gt 1.5\) 时，倾向形成
  • 离子键
  • 非极性共价键
  • 金属键
  • 氢键
\(\Delta X\gt1.5\) 时价电子几乎完全转移给电负性大的原子，形成正负离子和离子键，如 NaCl。【秒记】ΔX>1.5 → 离子键。
下列关于离子键的描述，正确的是
  • 有方向性和饱和性
  • 没有方向性和饱和性
  • 熔融后不导电
  • 硬度小、熔点低
离子键靠静电吸引、四面八方等同，故无方向性无饱和性；离子晶体硬度大、熔点高、熔融后能导电。共价键才有方向性和饱和性。【秒记】离子键：无方向、无饱和；共价键：有方向、有饱和。
Si 晶体中每个硅原子通过哪种杂化与周围 4 个原子成键，键角多少？
  • sp²，120°
  • sp³，109°28′
  • sp，180°
  • 不杂化，90°
Si 外层 \(3s^23p^2\) 形成 4 个等同 sp³ 杂化轨道，指向正四面体顶角，键角 109°28′，构成金刚石型结构。【秒记】Si=sp³=正四面体=109°28′。
石墨晶体中，层与层之间靠什么结合？
  • 共价键
  • 离子键
  • 范德华力（分子键）
  • 金属键
石墨层内是 sp² 共价键 + 自由 p 电子（金属性），层间相距 340 pm 靠较弱的范德华力，所以易滑移、可做润滑剂。【秒记】石墨层间=范德华力=易滑移。
晶体的基本性质包括（多选）
  • 自范性
  • 均匀性与各向异性
  • 对称性
  • 最小内能与固定熔点
课程归纳的晶体四大基本性质就是这四条。【秒记】四性质：自范、均匀/各向异、对称、最小内能固定熔点。
关于石墨是"混合键型晶体"，下列正确的有（多选）
  • 层内 C 原子以 sp² 共价键连成六角平面
  • 层内自由 p 电子使其能导电导热（金属性）
  • 层间靠范德华力结合（分子键）
  • 层间靠离子键结合
石墨同时存在共价键、金属键、分子键三种键，因此叫混合键型晶体；层间是范德华力而非离子键。【秒记】石墨=共价+金属+分子三键并存。
下列属于极性共价键化合物的有（多选，参考课程数据）
  • GaAs（\(\Delta X=0.4\)）
  • InSb（\(\Delta X=0.2\)）
  • GaP（\(\Delta X=0.5\)）
  • 金刚石（\(\Delta X=0\)）
\(0\lt\Delta X\lt1.5\) 为极性共价键，GaAs、InSb、GaP 都满足；金刚石 \(\Delta X=0\) 是非极性（典型）共价键。【秒记】化合物半导体多是极性共价键。
