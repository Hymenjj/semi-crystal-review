第一章　晶体的特性

考点 4 条 ·　打开第一章 →

1.什么是晶体？
内部质点（原子、离子、分子）在三维空间作周期性重复排列（长程有序）的固体。表现为有自范性（自发长成规则多面体）、固定熔点、各向异性、对称性、最小内能（最稳定）。与非晶体的根本区别＝有没有远程有序。 → 第一章 · 晶体 vs 非晶体（含 3D 演示）
2.晶体的基本性质？
均匀性、各向异性、自范性、固定熔点、对称性、最小内能（稳定性）。这些都源于内部质点的周期性有序排列。 → 第一章 · 晶体的基本性质
3.晶体的类型和结合力？
按结合键（结合力）分五类：离子晶体（离子键，如 NaCl）、原子（共价）晶体（共价键，如金刚石、Si）、金属晶体（金属键）、分子晶体（范德华力，如干冰）、氢键晶体（如冰）。键越强→熔点越高、硬度越大。 → 第一章 · 五种键型与结合力
4.石墨是混合价键型晶体——价键类型与性能的关系？
石墨同时含多种键：层内每个 C 以 sp² 共价键连成六角网（强）、还有离域 π 电子；层间靠较弱的范德华力。所以：层内强键＋π 电子 → 沿层面导电、强度高；层间弱 → 易解理滑移（软、润滑）。这就是“混合价键型晶体，价键类型决定性能”的典型例子。 → 第一章 · 石墨（含 3D 结构）

第二章　点阵理论

考点 6 条 ·　打开第二章 →

1.点阵和平移群：一维、二维、三维点阵？
点阵＝一组无限多、周期性排列的等同点（结点），每个结点周围环境完全相同；连接结点的所有平移操作构成平移群。按维数分一维（直线点阵）、二维（平面点阵）、三维（空间点阵）。结点 + 结构基元 = 晶体结构。 → 第二章 2.1 点阵和平移群
2.单位平行六面体（晶胞）的选取原则？
在能反映点阵周期性的前提下：① 尽量反映点阵固有的对称性；② 含直角（90°）数目尽量多；③ 在满足前两条下，体积尽量小。正因第③条，“底心立方”“面心四方”等会被化简成更小的已有格子而不独立存在。 → 第二章 2.2 / 不存在的格子如何并入更小格子
3.14 种布拉菲格子有哪些？为什么有些格子不存在？
7 大晶系，各自“在不破坏本晶系对称的前提下还能加哪种心（P 简单 / I 体心 / F 面心 / C 底心）”，合计 立方3＋四方2＋正交4＋六方1＋三方1＋单斜2＋三斜1 = 14 种。有些格子不存在，是因为加了某种心后：要么降低/不符合该晶系应有的对称，要么能化简成体积更小的已有格子（如底心立方→简单四方、面心四方→体心四方），按选取原则就不另算。 → 第二章 · 14 布拉菲格子（可切换）+ 格子并入演示
4.典型晶体结构要牢固掌握：NaCl、CsCl、ZnS、Si(Ge)、CaF₂、TiO₂、Cu₂O…作图题
逐个掌握其晶胞、配位数与代表：NaCl（Cl⁻面心立方、Na⁺填全部八面体空隙，配位 6）、CsCl（两套简单立方套构，配位 8，注意≠体心立方）、ZnS（闪锌矿=立方、纤锌矿=六方，配位 4）、Si/Ge（金刚石型）、CaF₂ 萤石（Ca 面心立方、F 填满四面体空隙，Ca 配位 8 / F 配位 4）、TiO₂ 金红石（四方，Ti 配位 6）、Cu₂O 赤铜矿（O 体心立方、Cu 直线配位 2）。本站这些都有可旋转 3D 晶胞，对着练画。 → 第五章 · 各结构 3D（NaCl/ZnS/金刚石/金红石/钙钛矿/萤石/赤铜矿）
5.能熟练画出晶胞，并清楚其结构基元、布拉伐格子、属于何点群？
对每个典型结构，要能说出：结构基元（重复单元里有哪些原子、占什么位置）、布拉伐格子（如 NaCl、金刚石、闪锌矿都是面心立方；CsCl 是简单立方）、所属点群（如金刚石 \(O_h\)、闪锌矿 \(T_d\)（无对称中心）、纤锌矿 \(C_{6v}\)）。配合作图题反复练手画晶胞。 → 第五章 · 结构基元 / 布拉伐格子 / 点群
6.三种描述晶体的方法：晶胞法、密堆积法、配位多面体法？
晶胞法：用平行六面体晶胞 + 原子坐标描述（最通用）。密堆积法：把大原子看作等径球作最紧密堆积（hcp ABAB / ccp ABCABC，利用率 74.05%），小原子填四面体/八面体空隙。配位多面体法：用中心原子与其配位原子构成的多面体（四面体、八面体…）及其连接方式描述。 → 第二章 2.4 密堆积 / 空隙 / 配位多面体

第三章　晶体的对称性理论

考点 4 条 ·　打开第三章 →

1.一些基本概念：等同图形、对称图形、对称动作、对称要素、阶次？
对称（动作）操作＝使图形复原（与自身重合）的动作；对称要素＝完成该操作所凭借的几何元素（点、线、面，如对称轴、镜面、对称中心）；等同图形＝由对称操作互相联系的全同部分；阶次＝某对称要素对应的对称操作的数目（如 n 次轴阶次为 n）。 → 第三章 · 基本概念
2.对称轴、反轴的轴次，只有 \(\bar4\) 是独立的，为什么？
反轴 = 旋转 + 反演的复合操作。其中 \(\bar1\)＝对称中心 i、\(\bar2\)＝镜面 m、\(\bar3\)＝3 次轴＋对称中心、\(\bar6\)＝3 次轴＋镜面（3/m）——它们都能用其它已有对称要素的组合来代替；唯独 \(\bar4\) 不能被任何其它对称要素的组合代替，所以反轴中只有 \(\bar4\) 是独立的。 → 第三章 · 反轴
3.宏观对称要素只有 32 种（32 点群），为什么？组合原则？
独立的宏观对称要素只有 \(1,2,3,4,6,\bar1(i),m,\bar3,\bar4,\bar6\) 这几种（受晶体学限制：只允许 1/2/3/4/6 次轴，没有 5 次及更高次轴）。把它们按组合定理（轴与轴、轴与面的夹角受限的组合规则）相互组合，独立的组合方式恰好是 32 种宏观对称类型 = 32 点群。 → 第三章 · 晶体学限制 / 32 点群
4.微观对称性有 230 种（空间群），为什么？组合原则？
在 32 点群（宏观对称）的基础上，结合 14 种布拉菲格子（平移），并引入只在微观才有的、带平移分量的对称要素——滑移面、螺旋轴，把这些再按组合规则组合，得到 230 种空间群。即：宏观 32 ＋ 平移（14 格子）＋ 滑移面/螺旋轴 → 微观 230。 → 第三章 · 空间群（230）

第四章　晶向、晶面等概念

考点 5 条（计算 + 作图主战场）·　打开第四章 →

1.晶面指数、晶向指数的规定，切莫混淆？
晶面指数 (hkl)：求三轴截距 → 取倒数 → 通分 → 取分子；圆括号 () 表一组平行晶面，花括号 {} 表晶面族。晶向指数 [uvw]：过原点平行线上某点坐标 → 化最简整数比；方括号 [] 表一晶向，尖括号 ⟨⟩ 表晶向族。别混：晶面取倒数、晶向不取倒数。 → 第四章 4.2/4.3（含 2D/3D 演示）
2.四方、六方晶系中，三个与四个密勒指数的关系和互换？
六方晶系常用四轴 \(a_1,a_2,a_3,c\)，晶面记 (hkil)，四个指数只有 3 个独立，满足 \(h+k+i=0\)（即 \(i=-(h+k)\)）；晶向记 [uvtw]，满足 \(u+v+t=0\)。三指数 ⇄ 四指数即按此关系补出/去掉第三个分量。 → 第四章 · 密勒-布拉菲指数
3.在四方、六方晶系中，根据密勒指数正确标出晶面和晶向？作图题
练习：给定 (hkl)/[uvw]，在晶胞图上按截距倒数画出晶面、按坐标画出晶向箭头；六方注意用四轴并满足 \(h+k+i=0\)。本站有可旋转的 3D 晶面指数演示，看晶面如何切过晶胞。 → 第四章 · 3D 晶面指数（看晶面切晶胞）
4.晶面夹角、晶向夹角、晶面间距等公式（立方晶系）？
晶面间距：\(d_{hkl}=\dfrac{a}{\sqrt{h^2+k^2+l^2}}\)（指数越低、间距越大）。两晶向/晶面夹角：\(\cos\varphi=\dfrac{h_1h_2+k_1k_2+l_1l_2}{\sqrt{h_1^2+k_1^2+l_1^2}\,\sqrt{h_2^2+k_2^2+l_2^2}}\)（立方同指数 \((hkl)\perp[hkl]\)，故晶面夹角同式）。晶带定律：\(hu+kv+lw=0\)。这些只对立方晶系这样简单。 → 第四章 4.5（含 4 道例题）
5.填图：(111)、(110)、(100) 三个方向立方晶系的标准投影？作图题
即极射赤面投影（标准投影）：把晶面取向投到赤道面上。立方晶系沿不同方向看到的对称花样：⟨100⟩ 取向 4 重、⟨110⟩ 取向 2 重、⟨111⟩ 取向 3 重对称（与第五章光学定向、反射对称呼应）。 → 第四章 4.6 极射赤面投影

第五章　典型半导体/电子材料的结构与性质

考点 4 条 ·　打开第五章 →

1.Si、Ge、GaAs 结构上的重要参数及与性能的关系？
Si、Ge＝金刚石型（同种原子、有对称中心 \(O_h\)）；GaAs＝闪锌矿型（两种原子、无对称中心 \(T_d\)）；配位数都是 4。要会看结构参数表：各晶面的面密度、价键密度、面间距（如金刚石结构 {111} 面间距最大 ≈0.43a、价键密度最低）——这些参数直接决定下面的解理/生长/腐蚀行为。 → 第五章 5.1/5.2（含面密度切片 3D）
2.掌握解理特性、生长特性、腐蚀特性，并解释相关现象？
解理：沿面间距大、价键密度小的面最易裂开——Si、Ge 解理面 {111}；GaAs、InSb 解理面 {110}。生长：理想外形由生长慢的低指数面包围，金刚石型趋向 8 个 {111} 面组成的正八面体。腐蚀：择优腐蚀出现蚀坑（{111} 三角形、{100} 方形、{110} 长方形），可用于定向、显错。GaAs 极性面 (111)Ga 稳定、\((\bar1\bar1\bar1)\)As 活泼。 → 第五章 5.2 解理/腐蚀/生长
3.典型的固溶体及其结构？
固溶体＝固态下一种组分溶解了其它组分形成的单一均匀结晶相。两类：替代（置换）式（溶质替代结点原子，分无序/有序/连续/有限等）、间隙（填隙）式（小原子 H/O/N/C/B 占空隙）。韦伽（Vegard）定律：替代式 \(a=(1-x)a_1+x a_2\)，晶格常数随成分线性变化。 → 第五章 5.4 固溶体
4.什么是液晶？什么是纳米晶？
液晶：某些物质在熔点与清亮点之间存在的中间态——既像液体能流动、又像晶体有各向异性。三类：近晶型（Smectic）、向列型（Nematic，用于显示器）、胆甾型（Cholesteric）。纳米晶：晶粒尺寸 1–100 nm，表面原子比例大，呈现量子尺寸效应、小尺寸效应、表面/界面效应、宏观量子隧道效应。 → 第五章 5.5/5.6 液晶与纳米晶

第六章　点缺陷

考点 3 条 ·　打开第六章 →

1.单质、化合物的点缺陷类型？浓度的估算？
类型：空位、间隙原子、置换原子；成对出现的 弗仑克尔缺陷（空位＋间隙，密度不变）、肖特基缺陷（正负离子空位成对、原子迁到表面，密度略降）。浓度估算：热平衡空位浓度 \(n=N e^{-u/kT}\)（由 \(\Delta F=n u_1-T\Delta S\)、\(S=k\ln W\) 推出）；弗仑克尔缺陷 \(n=\sqrt{N N^{*}}\,e^{-E_F/2kT}\)。 → 第六章 · 点缺陷类型 + 浓度公式
2.缺陷对晶体结构的影响？
点缺陷使周围点阵发生畸变（局部胀缩、产生内应力）；肖特基缺陷使晶体密度下降，弗仑克尔缺陷原子数守恒、密度基本不变；化合物中还涉及化学计量比偏离（非化学计量化合物）。 → 第六章 · 缺陷对结构的影响
3.点缺陷的电离对材料电学性能的影响？重点
这是本章重点：杂质（点缺陷）电离提供载流子，决定半导体的导电类型和载流子浓度。施主（5 价 P 取代 Si，多 1 个电子→ n 型，多子是电子）；受主（3 价 B 取代 Si，缺 1 个电子留空穴→ p 型，多子是空穴）。靠掺杂精确控制导电性，是一切半导体器件的基础。 → 第六章 · 施主/受主掺杂（含动画）

第七章　位错（线缺陷）

考点最多的一章 ·　最终目的＝位错对半导体性能的影响 ·　打开第七章 →

1.什么是刃（棱）型位错？螺型位错？混合型位错？作图题
刃型：晶体内多插入半个原子面，其边缘＝位错线，b ⊥ 位错线；半面侧受压、另侧受张（有正应力）。螺型：原子面被错开成连续螺旋面，绕位错线转一圈升高一个 b，b ∥ 位错线，无多余半原子面、无正应力。混合型：刃型 + 螺型的组合，b 与位错线成任意角。 → 第七章 7.1（刃型/滑移/螺型 3D）
2.什么是柏氏矢量？定义？求法？如何提出的？作图题
柏氏矢量 b＝表征位错畸变大小和方向的矢量。求法（弗兰克回路法）：① 在含位错的实际晶体里绕位错逐原子走一闭合回路；② 在完整参考晶体里按同样步数再走一遍，终点回不到起点；③ 从终点指向起点的矢量＝b。b 与回路大小、路径无关（守恒性）。刃型 b⊥线、螺型 b∥线。 → 第七章 7.2 柏氏回路（动画）
3.位错的产生及其密度的估算？
产生：机械应力（挤压/拉伸/切割/研磨）与热应力（温度梯度）超过屈服强度时产生。密度 ρ＝单位体积中位错线总长（或单位面积穿过的位错数），用蚀坑法等测量。 → 第七章 7.3 位错的产生
4.弯曲晶体 / 径向温度梯度造成的位错密度估算；位错增殖：L 型、U 型平面源？动画
弯曲晶体（曲率半径 r）：\(\rho=\dfrac{1}{r\,b}\)（如 Si 弯曲半径 1 m 时 \(\rho\approx2.6\times10^{5}/\text{cm}^2\)）。径向温度梯度：\(\rho=\dfrac{\alpha}{b}\left|\dfrac{dT}{dr}\right|\)。增殖（Frank–Read 源）：L 型平面源＝一端固定的位错线绕固定端卷成螺旋放环；U 型平面源＝两端固定的位错段弓出、过半圆失稳、箍断放出一圈圈同心位错环。 → 第七章 7.3 密度公式 + L/U 型平面源动画
5.位错的应力场和应变能；结论：应变能 ∝ 柏氏矢量的平方？应力场各参数意义？
应变能：\(W_{刃}=\dfrac{Gb^2}{4\pi(1-\nu)}\ln\dfrac{r_1}{r_0}\)、\(W_{螺}=\dfrac{Gb^2}{4\pi}\ln\dfrac{r_1}{r_0}\)——都 ∝ \(b^2\)（所以 b 小的位错更稳定，位错倾向取最小 b）。应力场：刃型 \(\sigma\sim\dfrac{D\cdot(\dots)}{r}\)，\(D=\dfrac{Gb}{2\pi(1-\nu)}\)（含正应力与切应力）；螺型只有切应力 \(\tau_{\theta z}=\dfrac{Gb}{2\pi r}\)、无正应力。参数：G 切变模量、b 柏氏矢量、ν 泊松比、r 到位错线距离。 → 第七章 7.4 应力场与应变能
6.位错与其它缺陷、位错与位错的相互作用？（交割、割阶；最稳定排列）
与点缺陷：刃型位错下方张应力区吸引大原子/杂质形成科垂尔气团（钉扎位错）：\(C=C_0 e^{-U/kT}\)。位错相遇：发生交割，在位错线上留下割阶（jog）/扭折（kink）。位错间力：\(F=\tau b\)；结论：同号刃型位错相互作用时，排在 X=0、沿 Y 轴一列（一根在另一根正上/下方）时最稳定——这正是小角晶界（位错墙）的来由。 → 第七章 7.5 相互作用 / 科垂尔气团
7.解释现象：小角晶界、金属的多边形化、位错排？
小角晶界＝取向差很小的两晶粒间，由一列等间距刃型位错构成，几何关系 \(D=b/\theta\)（θ 越大、位错排得越密）。多边形化＝弯曲晶体退火时，杂乱位错通过滑移＋攀移重新排成一道道位错墙（位错排），把晶粒分成多个小角度的亚晶——本质就是同号刃型位错沿 Y 排成列最稳定。 → 第八章 · 小角晶界 \(D=b/\theta\)（动画）
8.位错对半导体材料性能的影响？本章最终目的
位错是电活性的缺陷：① 引入深能级，成为载流子复合中心，降低少子寿命；② 位错线吸引杂质（气团），形成杂质沉淀、漏电通道，使 pn 结反向漏电增大、击穿变软；③ 影响迁移率、均匀性与器件成品率。所以制备高质量半导体追求低位错密度乃至无位错单晶——这就是研究位错的最终目的。 → 第七章 7.6 位错对半导体性能的影响

1.堆垛层错？不全（部分）位错？扩展位错？
堆垛层错＝密排面（fcc 的 {111}）堆垛顺序 ABCABC 在某处出错（抽出型/插入型；孪晶界＝从某层起镜像）的面缺陷，能量较低。不全（部分）位错＝柏氏矢量不等于点阵平移矢量的位错（如肖克莱、弗兰克部分位错），它界定层错区。扩展位错＝一个全位错分解成两个不全位错，中间夹一条层错带（因总能量降低而自发分解）。 → 第八章 · 堆垛层错（含演示）/ 扩展位错
2.小角晶界、孪晶界、相界等界面概念？
小角晶界：取向差小，由位错列构成，\(D=b/\theta\)（倾斜晶界＝刃型位错墙、扭转晶界＝螺型位错网）；θ 大到约 >10–15° 后位错重叠失效→成大角晶界。孪晶界：两部分晶体成镜像对称的界面（共格、能量低）。相界：不同相之间的界面（共格/半共格/非共格）。 → 第八章 · 小角晶界 / 孪晶 / 相界
